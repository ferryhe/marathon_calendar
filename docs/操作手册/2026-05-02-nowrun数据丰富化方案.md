# NowRun 数据丰富化方案

> 时间：2026-05-02
> 背景：NowRun（nowrun.cn）单条赛事详情页含有大量结构化数据，目前只用了 5% — 名称/城市/比赛日期/报名时间/状态。本文梳理可用字段、按价值分级，并给出落地路线。

## 二、字段优先级分级

### Tier A — 立刻有价值（建议第一批引入，schema 改动小）
1. **certification_grade** — A/B/C 类，前端可做"金标银标"徽章
2. **distance_options** (jsonb) — `[{kind:"full", capacity:8000, price:180}, ...]`，列表与详情都能用
3. **highlights** (text) — 赛事亮点段，能直接显示在赛事卡片首屏
4. **organizer** (text) — 主办单位
5. **start_location / finish_location / packet_pickup_location** (text)
6. **medal_image_urls** (text[]) — 奖牌图，详情页大图展示，超强视觉吸引
7. **race_time / registration_open_time / registration_close_time** (timestamp 或 time，已采 date 升级到 timestamptz) — 时分级精度

### Tier B — 提升体验（schema 改动中）
8. **city_guide** (jsonb) — `{must_eat, must_visit, transport}`，差异化内容、SEO 价值高
9. **lottery_history** (jsonb) — `[{kind:"full", year:2025, applicants:9556, hit_rate:0.837}]`
10. **official_documents** (jsonb) — `{registration_notice, race_rules, course_info, packet_pickup_guide}` 4 个链接
11. **official_wechat_account** (text)
12. **registration_channels** (text[]) — `["京视赛事小程序","数字心动APP"]`
13. **lottery_result_at** (timestamptz) — 出签结果时间

### Tier C — 锦上添花（数据量大，可以延后）
14. **weather_history** (jsonb) — 5 年同期天气
15. **course_description** (text) — 赛道长文描述
16. **course_image_url** (text) — 赛道图

## 三、Schema 变更草案

```sql
-- marathons 表追加（赛事级别"基本不变"的字段）
ALTER TABLE marathons
  ADD COLUMN certification_grade TEXT,            -- 'A'/'B'/'C'
  ADD COLUMN organizer TEXT,
  ADD COLUMN official_wechat_account TEXT,
  ADD COLUMN city_guide JSONB,                    -- {must_eat, must_visit, transport}
  ADD COLUMN weather_history JSONB;               -- [{year,date,low,high,condition,wind}]

-- marathon_editions 表追加（届次级别"每年都不同"的字段）
ALTER TABLE marathon_editions
  ADD COLUMN distance_options JSONB,              -- [{kind,capacity,price}]
  ADD COLUMN highlights TEXT,
  ADD COLUMN start_location TEXT,
  ADD COLUMN finish_location TEXT,
  ADD COLUMN packet_pickup_location TEXT,
  ADD COLUMN medal_image_urls TEXT[],
  ADD COLUMN race_start_time TIME,                -- 开跑时刻 HH:mm
  ADD COLUMN registration_open_time TIME,         -- 配合 registration_open_date
  ADD COLUMN registration_close_time TIME,
  ADD COLUMN lottery_result_at TIMESTAMPTZ,
  ADD COLUMN lottery_history JSONB,               -- [{kind,year,applicants,hit_rate}]
  ADD COLUMN registration_channels TEXT[],
  ADD COLUMN official_documents JSONB,            -- {notice,rules,course,pickup}
  ADD COLUMN course_description TEXT,
  ADD COLUMN course_image_url TEXT;
```

## 四、采集流程改造

NowRun 单页 HTML 结构稳定（标题 + 6 个 H3 区块 + 关键字段标签固定），可用纯 CSS/正则解析，不必上 LLM。

建议在 `server/sources/nowrun.ts`（新建）实现：
1. `parseNowRunRace(html: string): NowRunRaceData`，覆盖以上 Tier A+B 字段
2. `syncScheduler.ts` 中 nowrun 源的 fetch 后调用 parser，把字段写入 marathons + marathon_editions
3. `field_sources` 用 `source: 'nowrun'` 标记，方便后续多源融合优先级判定
