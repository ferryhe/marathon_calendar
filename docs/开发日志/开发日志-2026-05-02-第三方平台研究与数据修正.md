## 三、关键发现（避坑要点）

### 1. iranshao 爱燃烧已停服

测试了 10 个最热门马拉松的 `iranshao.com/races/{id}` 旧 URL（上海 583 / 北京 597 / 杭州 859 等），**全部 404**。爱燃烧主站现在仅剩业界资讯，赛事数据库已下线。

→ 不要再花时间搜 `site:iranshao.com/races`，SEO 索引尚未更新但点开都是死链。

### 2. shuzixindong 数字心动子域名不规律

猜测的 `{city}marathon.shuzixindong.com` 模式 8 个子域名：

| 子域 | DNS 状态 |
|---|---|
| `ningbomarathon` | ✅ 200 |
| `wuhanmarathon` / `qingdaomarathon` / `xianmarathon` / `chongqingmarathon` / `dalianmarathon` / `suzhoumarathon` / `wuximarathon` / `hengshuihumarathon` | ❌ DNS fail |

→ 不要批量假设，必须逐个 `curl` 验证。`race.shuzixindong.com` 主 SPA 需要 APP 登录态，不可爬。

### 3. 最酷的届次陷阱

`zuicool.com/event/{id}` 不区分届次。同一城市同一赛事，**不同年份是不同的 eventId**：

- 厦马 2026/01 在最酷有页面，但 ID 对应已结束的 2026 届，**不能绑给 2027 届**
- 宁马 2026/03 同理

→ 绑定前必须在页面上确认届次年份，未来同步任务也要重新发现新 ID。

### 4. 顶级赛事不一定上 mararun

mararun 覆盖北马/广马/深马/汉马/南马/成马等"田协 A1 类"赛事，但**自有报名平台的赛事不上 mararun**：

- 上马（shang-ma.com）
- 厦马（xmim.org）
- 杭马（hzim.org）
- 兰马 / 太马 / 重马（自有官网）

### 5. runchina 田协详情页不可爬

`portal.php?ac=detail&id={id}` 返回的 HTML 跟主页框架一致，**详细字段是 JS ajax 注入**，`webFetch` 拿不到。但首页"近期开赛"区块直接渲染了赛事列表，可作为兜底信息源。

→ 长期方案：每年 1 月解析田协官方发布的当年完整赛事目录 PDF（在 `file.shuzixindong.com/...`，hash 每年变）。
